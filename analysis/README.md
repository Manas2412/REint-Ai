# Forecast Analysis Notebook

This folder contains a Jupyter notebook (`forecast_analysis.ipynb`) that analyzes wind generation forecast error for January 2024.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Set your BMRS API key:

```bash
export BMRS_API_KEY=your_api_key_here
```

## Run

```bash
jupyter lab
```

Open `forecast_analysis.ipynb` and run the cells.
